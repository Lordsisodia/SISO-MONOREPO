# Landing Page Factory

Vertical playbooks are grouped for faster browsing:

- `hospitality/` — food, beverage, travel, experiential venues.  
- `wellness/` — health, beauty, and fitness businesses.  
- `professional-services/` — agencies, B2B services, trades, and white-collar pros.  
- `niche-experimental/` — specialty retail, creator economy, and emerging concepts.

When adding a new industry deck, drop it into the closest bucket and add a bullet here so searchers know it exists.
